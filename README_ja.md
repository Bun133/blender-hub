## BlenderHub

BlenderHubは、Blender向けバージョン管理ツールです。     
複数のバージョン、複数の設定のBlenderを登録でき、一覧からワンクリックで起動できます。

### 機能

- インストール済みのBlenderを登録する
- ポータブル版のzipファイルからインストールする
- バージョンごとに名前を付けて管理する
